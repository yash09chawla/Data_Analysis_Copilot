# 🤖 Data Analysis Copilot

An AI-powered Exploratory Data Analysis (EDA) tool that allows you to upload any CSV dataset and instantly receive visualizations, statistical summaries, and a professional narrative report generated by **Groq's LLaMA 3.3 70B** (which uses a 100% free API). 

---

## ✨ Features
- **Automated EDA**: Instantly calculates missing values, distributions, correlations, outliers, and categorical feature analysis.
- **AI-Generated Insights**: Uses LLaMA 3.3 70B (via Groq) to write a detailed, professional narrative explaining the key statistical findings and recommending next steps.
- **Downloadable Reports**: Generates a self-contained, beautifully styled HTML report that you can download and share with stakeholders.
- **Interactive Chat Q&A**: Ask follow-up questions about your data in natural language directly in the chat interface.
- **Target Variable Analysis**: Optionally specify a target column to get deeper insights regarding your ML target.

## 🚀 Getting Started

Follow these steps to set up and run the Data Analysis Copilot on your local machine.

### Prerequisites
- Python 3.10 or higher installed on your system.
- Git installed on your system.

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/data-analysis-copilot.git
cd data-analysis-copilot
```

### 2. Create a virtual environment (Recommended)
It's best practice to run Python apps in an isolated virtual environment so dependencies don't conflict with other projects.
```bash
# On macOS and Linux:
python3 -m venv venv
source venv/bin/activate

# On Windows:
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies
With your virtual environment active, install the required packages:
```bash
pip install -r requirements.txt
```

### 4. Get a FREE Groq API key
This project relies on the Groq API for lightning-fast AI generation.
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up (no credit card needed).
3. Navigate to **API Keys** and click **Create API Key**.
4. Copy your new API key.

### 5. Configure your environment variables
Create a `.env` file in the root directory of the project and add your Groq API key.
```bash
# In Mac/Linux terminal:
echo "GROQ_API_KEY=your_copied_api_key_here" > .env
```
Or manually create a file named `.env` in the project root and paste this line:
```env
GROQ_API_KEY=gsk_your_api_key_here
```

### 6. Run the application
Start the Gradio server:
```bash
python app.py
```
Then, open your web browser and navigate to: **http://localhost:7860**
